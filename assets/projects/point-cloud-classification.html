Overview

RGB Point Cloud Classification was my final project for MSAI 349: Machine Learning.

Point clouds are a common data format used in robotics and computer vision. In this project, we implemented a custom PointNet architecture and augmented it with RGB color on the point cloud to classify objects in a custom dataset we generated.

PointNet Architecture

PointNet is a deep learning architecture designed to process point clouds directly. It is invariant to input permutation and can learn local features from unordered point sets.

For our project, we implemented a custom PointNet architecture using PyTorch. The network was trained on a custom dataset of point clouds augmented with RGB color information.

RGB Augmentation

RGB color information provides valuable context for object classification in point clouds. By augmenting our dataset with RGB values, we enhanced the network's ability to distinguish between objects of similar shape.

During training, we concatenated the RGB values with the point cloud data and fed the combined input to the PointNet architecture. This allowed the network to learn color-based features in addition to geometric features.
